## What is Data Analytics (in simple terms)?
 
Let’s keep it real — Data Analytics is not as complicated as it sounds.
 
It’s simply the process of:
 
- 👉 Collecting data
- 👉 Understanding it
- 👉 And using it to make better decisions
For example, imagine a company noticing that their sales dropped last month. A data analyst will look into the data, identify patterns, and answer questions like:
 
- Why did sales drop?
- Which product performed poorly?
- What can be improved?
In short, **you turn numbers into meaningful insights.**

## Skills You Will Learn in Data Analytics
 
One of the biggest advantages of this field is that you don’t need to be a coding expert to start.
 
Here’s what you’ll typically learn:
 
- **Excel** — For data handling and basic analysis
- **SQL** — To work with databases
- **Power BI / Tableau** — For creating dashboards and visual reports
- **Basic Python** *(optional but powerful)* — For advanced analysis
These tools are practical, industry-relevant, and highly valued by employers.
